## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Access the app:
- Open your browser and go to `http://localhost:3000`
- Or scan the QR code shown in the terminal with your mobile device

## Usage

1. **Fill Bride Details**: Enter bride's name, nakshatra, date, and time
2. **Fill Groom Details**: Enter groom's name, nakshatra, date, and time
3. **Enter Matching Parameters**: Fill in the 8 main matching parameters (kootas)
