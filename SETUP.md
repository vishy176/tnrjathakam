# Setup Instructions

## Quick Start

1. **Install Dependencies** (Already done!)
```bash
npm install
```

2. **Start the App**
```bash
npm run dev
```

3. **Access on Mobile**
   - The terminal will show a network URL (e.g., `http://192.168.x.x:3000`)
   - Open this URL on your phone's browser
   - Or scan the QR code if shown

## For Mobile Testing

### Android
1. Open Chrome on your Android phone
2. Go to the network URL shown in terminal
3. Tap the menu (⋮) and select "Add to Home Screen"
4. The app will now work like a native app!

### iPhone
1. Open Safari on your iPhone
2. Go to the network URL shown in terminal
3. Tap the Share button (⎙)
4. Select "Add to Home Screen"
5. The app will now work like a native app!

## Using the App

1. **Fill the Form**
   - Enter bride and groom details
   - Fill matching parameters (kootas)
   - Add any additional information

2. **Generate PDF**
   - Click "WhatsApp లో పంపండి" button
   - The PDF will be generated automatically

3. **Share on WhatsApp**
   - On mobile: Use the native share dialog to send via WhatsApp
   - On desktop: PDF will download, then manually share

## Troubleshooting

**Can't access on mobile?**
- Make sure phone and computer are on the same WiFi network
- Check if firewall is blocking port 3000

**PDF not generating?**
- Make sure you filled in at least bride and groom names
- Check browser console for any errors

**WhatsApp share not working?**
- On desktop: PDF will download, share manually
- On mobile: Make sure WhatsApp is installed
